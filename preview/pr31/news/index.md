# Changelog

## rpt (development version)

- PR preview comments now use `recreate: true` to ensure they always
  appear at the bottom of the PR conversation, preventing them from
  being hidden in collapsed sections
  ([\#31](https://github.com/UCD-SERG/rpt/issues/31))

- Added RevealJS presentation format for Quarto vignettes and articles
  in pkgdown documentation. HTML pages now display an “Other Formats”
  section with links to slide versions
  ([\#29](https://github.com/UCD-SERG/rpt/issues/29))
